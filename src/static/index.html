Magane

Bringing LINE's stickers to Discord in a chaotic way.

Install Support GitHub

How does it work?

By installing Magane into your Discord client you gain access to a curated list of LINE sticker packs to use in the chat. It's as simple as that!

Now what?

After installing Magane, click the sparkling heart icon next to the emote list to bring up Magane's menu. By clicking on the little tools button you can access the full list of sticker packs supported and de/activate the ones you want to use.

You can also add stickers to your favorite list for rapid access by right clicking on them. This will keep them always on top!

Install options
  • BetterDiscord
  • Vencord

If you use BetterDiscord for all your Discord modding needs, then you can add Magane to it by downloading the following file and placing it in your plugins directory.

Download magane.plugin.js

If you are a Vencord user you need to download the plugin and compile Vencord yourself. You can find out how by following the guide on their GitHub.

Download magane.vencord.js
